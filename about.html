Project 1: EV Range Prediction System

  • Developed a range prediction system for electric vehicles (EVs). We proficiently handled real-world EV data from the BMW i3(60 ah), encompassing environmental, vehicle, battery, and heating circuit data.
  • Conducted Exploratory Data Analysis (EDA) to identify correlations and significant features affecting EV range. Employed data preprocessing techniques, including feature selection and normalization, to optimize model performance.
  • Utilized Multiple Linear Regression, Random Forest, and Deep Neural Network algorithms for EV range prediction and employed Recursive Feature Elimination (RFE) to identify crucial features for improved model accuracy.
  • Using the Random Forest algorithm, attaining an R-squared of up to 0.96. Produced models with a Mean Absolute Error (MAE) of 0.006, indicating precise range predictions.
  • Created a Digital twin model to simulate real-world EV range prediction scenarios and deployed the model on an AWS EC2 instance.
